PYTHON 透過雙語新聞情感分析進行台積電股價預測

在現今資訊爆炸的時代，金融市場受外部訊息影響的敏感度日益提高。台積電（TSMC）作為全球半導體產業的龍頭，其股價變動不僅受到基本面影響，亦高度依賴市場情緒、國際政治與產業新聞。然目前多數投資者與系統只關注技術分析與財報數據，忽略了「雙語新聞」所帶來的情緒波動對價格的潛在影響。 本研究嘗試整合中英文新聞情感分析，結合時間序列模型 SARIMAX，探索如何將「情感分數」作為外生變數納入股價預測模型中，進一步提升預測準確度，並提供投資決策輔助依據。
